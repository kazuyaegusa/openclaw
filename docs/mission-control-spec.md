# OpenClaw Mission Control 仕様書

> 出典: @gagarotai200 の X Article (2026-02-19)

## 概要

OpenClaw 単体は普通のAIエージェントだが、**Mission Control（管理画面）**を作ることで「自律型AIワーカー」に変わる。AIが自分で考えてタスクをこなし、その過程が全て可視化される。

## 技術スタック

| 項目 | 技術 |
|------|------|
| フロントエンド | Next.js |
| データベース | Convex |

**特徴**: リアルタイムデータ更新、軽快なUI。コードはOpenClawに指示して自動生成させる。

---

## 6つのコアツール

### 1. タスクボード（最優先）

AIが何をやっているかを可視化し、人間とAIの作業を一元管理する。

**カラム構成**:

| カラム | 説明 |
|--------|------|
| Todo | これからやるタスク |
| In Progress | 今やっているタスク |
| Done | 終わったタスク |

**担当者**: 自分 or AI

**プロンプト**:
```
タスクボードを作ってください。
すべてのタスクのステータスと担当者（私 or あなた）を
表示できるようにしてください。
今後、あなたが取り組むタスクは全てこのボードに登録し、
リアルタイムで更新してください。
```

---

### 2. カレンダー（2番目に重要）

AIのスケジュールタスク・Cronジョブを全て可視化する。

**表示内容**:
- スケジュールされたタスク
- Cronジョブ（定期実行タスク）
- 締め切り
- イベント

**プロンプト**:
```
ミッションコントロールにカレンダーを作ってください。
あなたのスケジュールされたタスクとCronジョブが全て表示されるようにしてください。
私がタスクをスケジュールするたびに、カレンダーに登録して、
正しく実行されているか確認できるようにしてください。
```

---

### 3. メモリ画面（3番目）

OpenClawのメモリを整理・検索可能なUIで表示する。

**UI構成**:
- メモリのリスト表示（日付順、カテゴリ別）
- 検索バー（キーワード検索）
- フィルター機能（日付、タグ、重要度）
- 詳細表示（クリックで全文表示）

**プロンプト**:
```
ミッションコントロールにメモリ画面を作ってください。
全てのメモリを美しいドキュメントとしてリスト表示してください。
また、全てのメモリを素早く検索できる検索コンポーネントも実装してください。
```

---

### 4. チーム画面

AIエージェントを「組織」として管理する。

**エージェント構成例**:

| エージェント名 | 役割 |
|---------------|------|
| メインAI（ボス役） | 全体の統括、タスクの割り振り |
| Developer | コード開発 |
| Writer | ライティング |
| Designer | デザイン制作 |
| Researcher | リサーチ・情報収集 |

**画面に表示する情報**:
- 各エージェントのプロフィール
- 担当領域
- 現在の作業ステータス
- 過去の実績

**プロンプト**:
```
チーム構造画面を作ってください。
あなた（メインエージェント）と、定期的に起動するサブエージェントを全て表示してください。
まだサブエージェントを考えていない場合は、作成して役割と責任で整理してください。
例として、開発者、ライター、デザイナーなどを含めてください。
```

---

### 5. コンテンツパイプライン（必要に応じて）

コンテンツ制作の全工程を自動管理する。

**カラム構成**:

| カラム | 役割 |
|--------|------|
| Ideas | アイデアのストック |
| Script | スクリプト作成中 |
| Thumbnail | サムネイル生成中 |
| Filming | 撮影待ち |
| Editing | 編集中 |
| Published | 公開済み |

**運用フロー**: アイデア投入 → AIが1日1回スクリプト自動生成 → サムネイル自動作成 → 撮影待ちへ移動

**プロンプト**:
```
コンテンツパイプラインツールを作ってください。
コンテンツ制作の全ステージを管理できるようにしてください。
アイデアの編集、フルスクリプトの記入、画像の添付ができるようにしてください。
このパイプラインを私と一緒に管理し、可能な限り追加作業をしてください。
```

---

### 6. オフィス画面

AIエージェントの動きをアバターで可視化する。

**機能**:
- 各エージェントのアバター表示
- 作業中 = 「PCに向かっている」状態
- 待機中 = 「アイドル」状態
- クリックで詳細ステータス確認

**プロンプト**:
```
デジタルオフィス画面を作ってください。
各エージェントが働いている様子を確認できるようにしてください。
エージェントは個別のアバターで表現され、それぞれの作業エリアとコンピューターを持っています。
作業中はコンピューターの前にいる状態にしてください。
全てのチームメンバーのステータスを素早く確認できるようにしてください。
```

## 実装手順

1. **OpenClawセットアップ** → インストール・起動
2. **初期設定プロンプト**:
   ```
   私のために「Mission Control」という名前のNextJSアプリを作ってください。
   データベースはConvexを使用してください。
   これから、タスク管理、コンテンツパイプライン、カレンダー、メモリ管理、
   チーム管理、オフィス画面の6つの機能を追加していきます。
   ```
3. **ツールを順番に追加**: タスクボード → カレンダー → メモリ → チーム → パイプライン → オフィス
4. **カスタマイズ**: 優先度カラム、タグ機能、パフォーマンス指標等を追加

## カスタムツール追加例

- 収益ダッシュボード（収益の可視化）
- SNS投稿スケジューラー（自動投稿）
- プロジェクト管理（複数プロジェクトの管理）
- 顧客管理（CRM機能）
- 分析ダッシュボード（データ分析）

## 注意事項

| 問題 | 解決法 |
|------|--------|
| AIが指示を忘れる | カレンダーに登録させる |
| タスクが進まない | タスクボードで進捗確認 |
| メモリが見つからない | メモリ画面で検索 |
| エージェントが動かない | チーム画面でステータス確認 |

- 最初はしっかり指示が必要。繰り返すうちにAIが学習する
- 週1回程度、古いタスクの削除やメモリの整理が必要
- Convex無料プランには制限あり（本格利用は月$25程度の有料プラン推奨）
